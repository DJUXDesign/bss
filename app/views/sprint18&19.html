Sprint 18 & 19 test scenarios

Scenario 1
Description

Wife reporting death of husband

  • Passes security (About caller)
  • Estimated date of death (About deceased)
  • Shows non-verified on header(Hospital)
  • Shows non-verified on CYA
  • No DAP (think it will be the solicitor – not confirmed yet)
  • BSP Start page
  • Goes into BSP (Bailout page)
  • Skips FEP too upset to apply (FEP Start page)
  • No declaration – no DAP or BSP or FEP
Scenario 2
Description

Cousin reporting death of cousin

  • Only knows name and first line of address – nothing else (fails security, shows non-verified on CYA)
  • Knows date of death(About deceased)
  • Getting ESA
  • Not eligible for BSP (BSP Start page)
  • Not eligible for FEP (FEP Start page)
  • No declaration – no DAP or BSP or FEP – (New declaration page)
Scenario 3
Description

Girlfriend reporting boyfriend’s death

  • Passes security (About caller)
  • Knows date of death (About deceased)
  • Getting JSA
  • Shows non-verified on header(Hospital)
  • DAP – yep, that will be me, give bank details
  • BSP – not eligible (BSP Start page)
  • FEP – yep, I’ll be arranging (FEP Start page)
  • Bail out of FEP on 2nd question(FEP Bailout page)
  • Declaration- would be there for DAP (New declaration page)
